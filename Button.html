Design System

Overview

Buttons are interactive elements that initiate a certain action when clicked. They serve as instructions for the user, giving them the ability to interact with the page in various ways. The text on the button informs the user of the action that will happen when they click it.

When to use

It is recommended to utilize buttons to communicate actions to users and to enable interaction with the page. Each page should have a prominent button and any additional actions should be represented with buttons of lower importance.

When not to use

Avoid using buttons as a means of navigation. For navigation purposes, use links instead when you want to direct the user to a different page.
