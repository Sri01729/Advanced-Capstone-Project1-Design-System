Design System

Overview

In a design system, checkboxes are used as a way to allow users to indicate a binary choice or multiple choices. Some common examples include selecting preferences, acknowledging terms and conditions, or marking items as complete.

When to use

When you need to allow users to select multiple options from a set of choice, only if the choices are independent and do not affect each other And also When the number of options are low.

When not to use

When you need users to enter a numerical value and if the options are not independent and affect each other then use radio buttons. Also if the number of options is large, consider using a drop-down menu or a select component.


It's important to consider accessibility when using checkboxes, ensure that their purpose and state are clear to all users, including those using assistive technology.
