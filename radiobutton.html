Design System

Overview

A radio button is a graphical element that allows a user to choose one option from a set. When a radio button is selected, it is highlighted and cannot be deselected until another option is chosen. Radio buttons are often used in forms and surveys, where multiple options are presented and the user must choose only one.

When to use

Radio buttons are a good choice when you want to:

  • Present a list of options where only one can be selected
  • Show that only one option can be selected at a time
  • Offer a clear visual representation of available options

When not to use

Radio buttons may not be the best choice when:

  • The user needs to select more than one option.
  • The options are not mutually exclusive
  • The options need to be presented in a different format, such as a dropdown menu or a list of checkboxes


Overall, radio buttons are a straightforward and effective component in a design system, but it's important to consider the context and options being presented before choosing to use them.
