Design System

Overview

A navigation system typically includes dropdown menus, links, buttons, and other interactive elements that allow users to access different sections of a website or application. Navigation design should consider the hierarchy of information and tasks within the system, as well as the user's mental model and expected behavior.

When to use

When your website or application has multiple sections or pages that need to be organized and accessed easily by users.

When consistency in navigation is important for the user experience and branding.

When the navigation can help users find the information they need quickly and easily.

When not to use

When your website or application only has a few pages or sections that don't require much organization.

When the navigation could add unnecessary complexity or confusion to the user experience.

When there are more effective ways to organize and access the content, such as search or filtering.


In summary, navigation is an important component of a design system that should be carefully considered in terms of user needs and the structure of the system. When used effectively, navigation can improve the user experience and make it easier for users to find the information they need.
