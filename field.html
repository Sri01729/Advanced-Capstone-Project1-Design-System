Design System

Overview

Input fields are crucial in any website for better communication with the users. They allow users to enter data such as text, numbers, or selections.

When to use

Input fields should be used when the interface requires user input, such as filling out a form, entering a password, or making a selection. They are an essential component of any user-driven interaction.

When not to use

Input fields should not be used when the information being presented is purely informational, such as a display of the user's name or account balance. In these cases, a simple display element is more appropriate.


In a design system, it's important to ensure that input fields are consistent and intuitive to use. They should also adhere to accessibility guidelines and be designed in a way that is user-friendly and easy to understand.
