Design System

Overview

Video can be an important component of a design system. It can be used to enhance user experiences, communicate information and create engagement. However, including video in a design system requires thoughtful consideration of the design principles, user needs, and technical requirements.

When to use

Video can be a useful tool in a design system when it adds value to the user experience. Here are some scenarios where video can be effective:
  • To demonstrate how a product or feature works: A short video can be a great way to showcase how a product or feature works, especially if it involves complex interactions or animations that are difficult to convey through static images or text.
  • To provide guidance or instructions: Video can be an effective way to provide step-by-step instructions or guidance, especially for complex processes.
  • To create engagement: Video can be a powerful tool for creating engagement and emotional connections with users. For example, a video can be used to showcase the values or mission of a brand.

When not to use

While video can be a useful tool in a design system, there are also scenarios where it may not be the best choice. Here are some examples:
  • When it doesn't add value: Video should only be used if it adds value to the user experience. If it doesn't add anything meaningful, it may be better to use a simpler, more lightweight solution.
  • When it's too resource-intensive: Videos can be large files that take up a lot of bandwidth and slow down the user experience. If the video is too resource-intensive, it may be better to use a simpler solution like text or images.
  • When it's not accessible: Video can be difficult for users with certain disabilities to access. For example, users with hearing impairments may not be able to access the audio in a video. If the video isn't accessible, it may be better to use a different solution that is more inclusive.

Overall, video can be a valuable component of a design system when used appropriately. As with any design element, it's important to consider the user's needs and ensure that the video adds value to the overall experience.
