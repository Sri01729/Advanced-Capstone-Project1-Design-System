Design System

Overview

"Align" in a design system refers to the positioning of elements within a design layout to create a visual balance and hierarchy. This can be done through adjusting the margins, padding, and spacing between elements, and using consistent alignments such as center, left, or right.

When to use

  • To create visual harmony and consistency in a design
  • To direct the user's attention towards important elements
  • To reinforce the hierarchy of information in a layout

When not to use

  • When the design needs to be more organic or free-flowing
  • When the elements don't fit within the established alignments
  • When the elements need to be placed in specific locations for functional reasons, such as buttons being placed in a consistent location throughout a website.

It's important to use alignments in a thoughtful and intentional way to enhance the user experience, while also maintaining a cohesive and aesthetically pleasing design.
