Design System

Overview

Flexbox works by defining a parent container as a flex container and its direct children as flex items. The layout of the flex items can then be controlled using CSS properties such as flex-direction, justify-content, and align-items. Flexbox is particularly useful for creating flexible, responsive layouts that can adapt to different screen sizes and devices.

When to use

Flexbox is a good choice when:

  • You want to arrange elements in a row or column.
  • You need to control the size and position of elements based on the available space.
  • You need to create responsive layouts that adjust to different screen sizes.

When not to use

Flexbox may not be appropriate when:

  • You need to create complex, multi-dimensional layouts.
  • You need to support older browsers that do not have full support for Flexbox.


Overall,Using Flexbox, we can easily control the placement of these elements and ensure they take up the full width of the container while maintaining a defined height.

                .flexelement {
                width: 200px;
                height: 100px;
                text-align: center;
                }
                </style>

                <div class="flexblock">
