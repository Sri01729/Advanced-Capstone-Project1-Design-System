Design System

Overview

A grid system in design refers to a structure consisting of vertical and horizontal lines that are used to arrange content on a page. It provides a framework for consistent placement of elements, making the design process easier and improving the overall visual hierarchy and balance of the design. A well-designed grid system can help create a harmonious, organized, and aesthetically pleasing layout.
