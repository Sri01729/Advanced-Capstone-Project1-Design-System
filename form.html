Design System

Overview

Forms are a fundamental component of any design system and can include a variety of input fields, such as text boxes, drop-down menus, radio buttons, checkboxes, and more. They are used to gather information from users, such as personal information, preferences, or feedback.

When to use

Forms should be used when the interface requires the collection of information from the user, such as when creating an account, making a purchase, or providing feedback.

When not to use

Forms should not be used when the interface does not require user input or when there is no need to gather information from the user. In these cases, a simple display element or a different interaction method may be more appropriate.


In a design system, forms should be designed to be intuitive and user-friendly, with clear labels and instructions, consistent formatting, and accessible to all users. It's also important to ensure that forms are secure, protecting the user's sensitive information and preventing data breaches.
